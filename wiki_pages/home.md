## üéØ About StorCat

StorCat is a modern Electron-based directory catalog management application that provides a clean, intuitive GUI interface for creating, searching, and browsing directory catalogs. Built with React 18 and TypeScript, StorCat maintains 100% compatibility with catalog files created by the original `sdcat.sh` bash script while offering enhanced functionality through a modern interface.

### Key Features

- **üìÅ Create Catalogs**: Generate comprehensive JSON and HTML catalogs of any directory
- **üîç Search Catalogs**: Fast, full-text search across multiple catalog files
- **üìÇ Browse Catalogs**: Interactive catalog browser with modern table interface
- **üåà Advanced Theming**: 11 beautiful themes including Dracula, Nord, Solarized, and more
- **üìê Flexible Layout**: Configurable sidebar positioning with intelligent icon placement
- **‚ö° Cross-Platform**: Works seamlessly on Windows, macOS, and Linux
