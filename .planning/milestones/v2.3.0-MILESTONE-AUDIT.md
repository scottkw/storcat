### Critical Cascade Finding

The Windows signing gap doesn't just block Windows artifacts — it blocks the **entire release pipeline**:

```
build-windows fails (no secrets) → release job blocked (needs: all 4 builds)
→ release stays draft → distribute.yml never fires
→ Homebrew tap NOT updated → WinGet NOT updated
```

**Mitigation options:**
1. Store the 4 eSigner secrets (resolves everything)
2. Add `if: success() || ...` conditional to release job to allow partial releases (temporary workaround)
3. Skip Windows build job entirely until secrets are provisioned
