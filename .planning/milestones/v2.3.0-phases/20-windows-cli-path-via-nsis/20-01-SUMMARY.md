## Decisions Made

- **EnVar plugin over EnvVarUpdate macro:** NSIS wiki explicitly warns that EnvVarUpdate silently corrupts PATH values longer than NSIS_MAX_STRLEN (1024 bytes default). EnVar uses Win32 `RegQueryValueEx` directly — no length limit. Developer machines with many tools (JDK, Python, Go, Node) easily exceed 1024 bytes.
- **Bundle EnVar.dll in repo:** Reproducible builds with no CI network dependency. MIT-licensed binary is only 9KB. Alternative (CI download step) would add complexity and a network failure point.
- **`!addplugindir "${__FILEDIR__}\Plugins"` form:** Uses NSIS 3.x predefined constant for the script's own directory, avoiding ambiguity between makensis working directory and script location.

None — no external service configuration required.

**Windows UAT checklist (manual, Windows machine required):**
1. Install StorCat via built NSIS installer
2. Open a NEW Command Prompt or PowerShell window
3. Run `storcat version` — should print version info
4. Check System Properties > Environment Variables > System PATH contains StorCat install directory
5. Uninstall StorCat
6. Verify StorCat install directory removed from System PATH
