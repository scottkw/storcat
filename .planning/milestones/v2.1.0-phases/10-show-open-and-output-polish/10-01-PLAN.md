    Then create cli/show.go replacing the stub in stubs.go. Implementation:

    1. **runShow(args []string) int** — follows interspersed flag pattern from list.go:
       - FlagSet "show" with ContinueOnError
       - Flags: `--json` (bool), `--depth` (int, default -1), `--no-color` (bool)
       - Usage func prints: "Usage: storcat show <catalog.json> [flags]\n\nDisplay a catalog's tree structure.\n\nFlags:\n  --depth N   Limit tree depth (-1 = unlimited, 0 = root only)\n  --json      Output raw catalog JSON\n  --no-color  Disable color output\n"
       - Separate positional from flag args (same pattern as list.go)
       - Parse flags; if ErrHelp return 0
       - Validate: len(positional) < 1 -> stderr "storcat show: catalog file argument required\nUsage: storcat show <catalog.json> [flags]\n", return 2
       - Validate: !strings.HasSuffix(positional[0], ".json") -> stderr "storcat show: expected a .json catalog file\n", return 2
       - If *noColorFlag: set `color.NoColor = true`
       - Load catalog: `svc := search.NewService(); root, err := svc.LoadCatalog(catalogPath)` — on error: stderr "storcat show: %v\n", return 1
       - If *jsonFlag: return `printJSON(root)`
       - Print root line: `fmt.Fprintf(os.Stdout, "%s\n", colorDirName(filepath.Base(root.Name)))` where colorDirName uses the dirColor printer
       - For each child in root.Contents: call `printTree(os.Stdout, child, isLast, "", 0, *depthFlag)`
       - Return 0

    2. **printTree(w io.Writer, item *models.CatalogItem, isLast bool, prefix string, currentDepth, maxDepth int)** — recursive tree walker:
       - connector: "├── " if !isLast, "└── " if isLast
       - name: `filepath.Base(item.Name)`
       - If item.Type == "directory": name = `dirColor.Sprint(name)` where dirColor is `color.New(color.FgBlue, color.Bold)` created once at package level
       - Print: `fmt.Fprintf(w, "%s%s%s\n", prefix, connector, name)`
       - If item.Type == "directory" and len(item.Contents) > 0:
         - If maxDepth >= 0 && currentDepth >= maxDepth: return (depth limit reached)
         - newPrefix: if isLast -> prefix + "    " else prefix + "│   "
         - For each child: printTree(w, child, childIsLast, newPrefix, currentDepth+1, maxDepth)

    3. Package-level var: `var dirColor = color.New(color.FgBlue, color.Bold)` — created once, not per-call

    4. Imports: "flag", "fmt", "io", "os", "path/filepath", "strings", "github.com/fatih/color", "storcat-wails/internal/search"
